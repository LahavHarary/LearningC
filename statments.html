פעולות שניתן להשתמש בהן בשפת סי לצורך בדיקה בקרה:
1. > (קטן / גדול מ..)
2. < (קטן / גדול מ..)
3. >= (קטן שווה / גדול שווה מ..)
4. <= (קטן שווה / גדול שווה מ..)
5. == (שווה)
6. != (שונה)

משפטים בשפת סי שניתן להשתמש בהן לצורך בדיקת בקרה:
if() - פעולה שבודקת האם מה שבתוך הסוגריים מתקיים - אם כן אנחנו נכנסים לתוך הקוד שימצא בתוך סוגריים מסולסלות.

else - פעולה שנעשית אם ורק אם פעולת הif שקדה לה לא התקיימה - גם פה , במידה ואנחנו מגלים שה else התקיים אנחנו נכנסים לתוך קטע הקוד שימצא בתוך סוגרים מסולסלות.

else if() - זוהי בדיקה שעובדת לאחר שיש if לפניה. הבדיקה תיעשה אם ורק אם הif הקודם שלנו לא התקיים.

נראה דוגמה במילים ולאחר מכן נפתור אותה בכתב:
נניח שאנחנו בודקים האם גילו של אדם הוא מספיק מבוגר כדי לעלות על מתקן בלונה פארק ונניח שגילו המינימלי של אדם שעולה על המתקן צריך להיות 15.
ז"א - כדי שאדם יוכל לעלות על המתקן אנו נצטרך לוודא שהוא לפחות בן 15.

פתרון בקוד: #include void main() // main function - this is the code that will be executed when we click f5. { int age = 15; // declaring an integer variable named age and setting in to 15. if(age >= 15) // checking if our age variable is greater or equal to 15. { printf("You can enter\n"); // this code will be executed only if our if statement is true. } else // the code inside the curly brackets will be executed only if our if statement is false. { printf("You can't enter\n"); } } // End of main function
