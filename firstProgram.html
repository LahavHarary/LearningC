Learning C

התוכנית הראשונה שלנו

בתוכנית זו נלמד איך מדפיסים למסך hello world:
include = פעולה שאומרת בעצם איזה ספרייה לכלול בתוך התוכנית (במקרה שלנו את הספרייה הכללית)
stdio.h = הספרייה הכללית (נמצאת בכל תוכנית שלנו)
void main() = התוכנית הראשית שלנו – כל הקוד שירוץ בתוכנית שלנו יתחם על ידי סוגריים מסולסלים
printf("") = פעולת הדפסה בסיסית – נרשום את מה שאנו רוצים להדפיס בתוך הגרשיים שבתוך הסוגריים.
