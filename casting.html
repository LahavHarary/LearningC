המרה - קסטינג

קסטינג = המרה בין משתנה למשתנה.
לפעמים אנחנו נרשום במשתנה מסוים ונרצה להמיר אותו למשתנה אחר בהמשך הדרך (או למשל לעבוד במשתנה מסוים ולהדפיס את התוצאה במשתנה אחר).
לדוגמה:
double num = 0; printf("please enter a number/n"); scanf_s("%lf",&num); printf("your number without decimal point is: %d", (int)num);
מה שעשינו פה:
ביקשנו מהמשתמש מספר ושמרנו אותו בתוך משתנה num מסוג double.
לאחר מכן הדפסנו את המספר שלו ללא הנקודה העשרונית בעזרת שימוש בcasting.

casting עובד בצורה הבאה:
לקחנו את המשתנה שלנו num שהוא מסוג double ובחרנו להציג אותו / להמיר אותו זמנית לint כדי להציג את המספר ללא הנקודה העשרונית.
ניתן לבצע המרה בצורה רגעית כמו שביצענו עכשיו כלומר, num אינו מושפע מהתהליך אלא רק מודפס בצורת int ולאחר מכן חוזר לעצמו.
ניתן גם לבצע המרה באופן קבוע למשל:
double num = 8.325; int new_num = (int)num;
במקרה הזה בעצם ביצענו המרה קבועה מdouble לint ושמרנו אותו בnew_num.
