משתנים

משתנים הינם עצמים שמטרתם היא לאחוז בסוגי נתונים שונים.
כל סוג משתנה טופס סוג שונה של מקום בזיכרון ויכול להחזיק נתונים שונים.

int

משתנה מסוג int הוא משתנה שתופס 4 בתים בזיכרון והוא מכיל מספרים שלמים כמו 1,2,3,4,5 וכו'.
הצהרה - כדי ליצור משתנה מסוג int אנחנו צריכים לרשום את ההצהרה הבאה:

הסבר:
int – אומר בעצם למחשב שעכשיו אנחנו יוצרים משתנה שיכול להכיל מספרים שלמים.
name – זה השם של המשתנה.

ישנה דרך נוספת להצהיר על משתנה int וזה על ידי signed/unsigned וההצהרה שלהם תהיה בצורה הזאת:


כאשר אנחנו רושמים signed int אנחנו אומרים למחשב שזה משתנה מסוג int אשר יכול להכיל גם מספרים חיובים וגם מספרים שלילים.
כאשר אנחנו רושמים unsigned int אנחנו אומרים למחשב שזה משתנה שיכול להכיל רק מספרים חיובים.
טווח מספרים- כאשר אנחנו משתמשים ב int טווח המספרים שלו נע בין מינוס 231 לבין 231 פחות אחד.
וכאשר אנו משתמשים במשתנה מסוג unsigned int טווח המספרים שלו נע בין אפס לבין 232.
