משתנים

משתנים הינם עצמים שמטרתם היא לאחוז בסוגי נתונים שונים , כל סוג משתנה טופס סוג שונה של מקום בזיכרון ויכול להחזיק נתונים שונים.

int
משתנה מסוג int הוא משתנה שתופס 4 בתים בזיכרון והוא מכיל מספרים שלמים כמו 8,9,12, 12345 וכו'
הצהרה - כדי ליצור משתנה מסוג int אנחנו צריכים לרשום את ההצהרה הבאה:

int – אומר בעצם למחשב שעכשיו אנחנו יוצרים משתנה מסוג int שיכול להכיל מספרים שלמים.
name – זה השם של המשתנה ומסוג int.
ישנה דרך נוספת להצהיר על משתנה int וזה על ידי signed unsigned וההצהרה שלהם תהיה בצורה הזאת:


כאשר אנחנו רושמים signed int אנחנו אומרים למחשב שזה משתנה מסוג int אשר יכול להכיל גם מספרים חיובים וגם מספרים שלילים.
כאשר אנחנו רושמים unsigned int אנחנו אומרים למחשב שזה משתנה שיכול להכיל רק מספרים חיובים.
טווח מספרים- כאשר אנחנו משתמשים ב int טווח המספרים שלו נע בין -2,147,483,648 ל 2,147,483,647 וunsigned int נע בין0 ל 4,294,967,295
חשוב לציין ש int name שקול להצהרה של signed int name לכן טווח המספרים כאן זהה
