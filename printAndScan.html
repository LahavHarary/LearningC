קלט ופלט

בשפת סי יש לנו אופציה להדפיס למסך / למשתמש שלנו דברים באמצעות פקודת printf() (שעליה הסברנו ב"תוכנית הראשונה שלנו").
בנוסף יש לנו אופציה להדפיס משתנים (שעליהם הסברנו "משתנים") בצורה הבאה:
printf("%(kind)",name_of_variable);
הסבר:
printf("%left",right) = פקודת ההדפסה שלנו – במקרה שלנו אנחנו רושמים בצד שמאל % ואת סוג המשתנה שאותו אנו מדפיסים ובצד ימין אנו רושמים את שם המשתנה שאותו אנו מדפיסים.
לדוגמה:
#include void main() { int x = 7; printf("%d",x); }
printf = הפקודה שאנו מבצעים
"%d" = סוג המשתנה שאותו אנו מדפיסים (במקרה שלנו אינט)
x = שם המשתנה שאותו אנו מדפיסים
אפשר לחשוב שבאמצעות הקוד שלנו אנחנו בעצם אומרים – תדפיס את הערך של int בשם x.
קלט:
בשפת סי יש אופציה לקלוט מידע מהמשתמש שלנו באמצעות פקודת scanf() שעליה נסביר:
scanf("%kind", &name_of_variable);
הסבר:
פקודה שבה אנו מכניסים ערך לתוך המשתנה שנמצא בצד ימין, גם פה אנו צריכים להגיד מאיזה סוג משתנה אנחנו מקבלים מידע scanf("%left",&right); .
לדוגמה:
#include void main() { int x; printf("Please enter a value"); scanf("%d",&x); printf("The value is: %d",x); }
ברגע שנריץ את התוכנית הזו הקונסול שלנו ימתין עד שניתן לו ערך.
לאחר שנכניס ערך יודפס לנו:
