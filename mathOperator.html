אופרטורים חשבוניים

  • + פעולת חיבור.
  • / פעולה חילוק.
  • - פעולת חיסור.
  • * פעולת כפל.
  • % מודולו - אופרטור שבעצם אומר לנו מהי שארית החלוקה של מספר במספר אחר.
  • () סוגריים.
  • = השמה - זוהי בעצם פעולה שאומרת בתכנות שמה שנמצא בצד שמאל מקבל את הערך שנמצא בצד ימין.
דוגמה למודולו:
7%3 = 1
כי:

2 1⁄3 = 7⁄3


כדי להגיע לשארית החלוקה אנו נכפיל את השבר שנשאר לנו ( 1⁄3 ) בערך שחילקנו בו את המספר ( 3 ).
מהי שארית החלוקה של 12 ב4 ?
באותה הדרך אנו נעשה קודם כל :
12⁄4 = 3.0
ניקח את ה 0⁄4 שיש לנו ובעצם שארית החלוקה שלנו היא 0.
באופן כללי כדי לחשב מודולו של מספר אנו נפעל לפי הנוסחא הבאה:
x - ( ( x⁄y ) * y) כך ש x הוא המספר ההתחלתי
ו y הוא המספר עליו אנחנו רוצים לעשות עליו את המודולו.

דוגמה להשמה:
בעצם מה שקורה פה זה שאנחנו מצהירים על משתנה בשם x ואנחנו נותנים לו את הערך 5, כלומר עשינו פה השמה.
במקרה הזה, לקחנו את x (שכבר מוכר לנו כ-int), וביצענו עליו השמה של הערך 7.
כאשר אנו עושים השמה, הערך שהיה במשתנה לפני "נדרס".
כלומר, אחרי שורה 5 יש במשתנה x רק את הערך 7.
